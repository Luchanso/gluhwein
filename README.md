# Luchanso Starter Kit

On board:
* Gulp
* ES6
* Babel
* Phaser

Install:
```sh
git clone https://github.com/Luchanso/starter-kit.git
cd starter-kit
npm i
npm i gulp -g
```

One line install:
```sh
git clone https://github.com/Luchanso/starter-kit.git && cd starter-kit && npm i && npm i gulp -g
```
